# SDK Examples

These examples show how external TypeScript consumers can call a running `codebase-docs-ai` API service through `@codebase-docs-ai/sdk`.

Examples:

- [node-generate-from-archives.ts](./node-generate-from-archives.ts): Node.js archive upload and single Markdown download.
- [browser-upload-file.ts](./browser-upload-file.ts): browser `File` upload from a file input.
- [custom-polling-and-download.ts](./custom-polling-and-download.ts): manual lifecycle control with custom polling.
- [error-handling.ts](./error-handling.ts): `CodebaseDocsAIClientError` handling.

These files are copy-paste examples, not a compiled package. Adjust imports, paths, and output handling for the host application.
